Coronavirus open citations visualisation

Showing works published in the past 20 years using open data by OpenCitations and Crossref

The visualisation

The following visualisation shows the most relevant articles investigating coronaviruses in the past 20 years, considering those published in three distinct periods characterised by three primary related diseases: SARS, MERS, COVID-19. The more an article is close to the centre of the circle shown, the more it is linked with the other articles. Clicking on an article will show its metadata on the right and all the other articles related to it through citations.

How it works

We developed a Python notebook to retrieve all the data used in the visualisation. We got a list of relevant articles talking about coronaviruses using the Crossref API. We looked for all the articles which contain the word coronavirus either in their title or abstract (total: 9,087 articles, as of 12 April 2020). Then, we retrieved all the citations which involve these articles, either as a citing entity or as a cited entity. We used the unifying REST API for all the OpenCitations Indexes (total: 124,295 citations, as of 12 April 2020). Finally, we used the Crossref API again to identify all the basic metadata (i.e. authors, year of publication, title, publication venue, and DOI) of the articles involved in all the citations retrieved (total: 42,213 articles, as of 12 April 2020). In the case Crossref did not return any metadata for some article, we completed the related metadata hand.

We showed only a partial subset of the articles and the related citations in the visualisation, mainly for computational purposes. Indeed, it would be too much expensive to create a diagram which includes all the articles and citations retrieved, which would make unclear and unusable the visualisation. In particular, we considered only the citations having both the citing entity and the cited entity included in the set of relevant articles talking about coronaviruses retrieved through the Crossref API. Also, we visualised only the articles that received, overall, at least ten citations per year since their publication date. We obtained 560 citations between 93 articles.

We created the visualisation using Cytoscape JS and JQuery. We used Pure CSS to define the layout of the website.. All the software, data, and additional material used for creating this website and the related data are available in our GitHub repository.

The data

The full set of data retrieved by running the notebook, both citations and article metadata, have been stored in JSON format compatible with the one used by Cytoscape JS for visualising data. The data are available on Zenodo:

Peroni, S. (2020). Coronavirus open citation data. Version 1.0. Zenodo. https://doi.org/10.5281/zenodo.3756803

We released all the data in CC0 to enable anyone to use them for any purpose.

Currently, the data do not include articles published in 2020, since the last release of COCI, the OpenCitations Index of Crossref open DOI-to-DOI citations, which is the most significant dataset of open citation data released by OpenCitations, includes citations of articles published until October 2019. We will release new versions of these data when future releases of COCI will be available, or when new citation data will be published in CROCI, the Crowdsourced Citation Index.

Help us

Currently (as of 12 April 2020) there are no articles published in 2020 (i.e. about COVID-19) included in the visualisation and the full data released, due to the absence of citations involving any of these articles in the OpenCitations Indexes. Of course, the situation will improve once the new release of COCI will be published.

However, even considering future releases of COCI, we will still miss several relevant citations such as those coming from articles of publishers that are not participating to the Initiative for Open Citations (I4OC) or those coming from preprints that do not also deposit their citation data in Crossref. You can help in improving this situation by sharing citation data in CROCI.

CROCI, the Crowdsourced Citation Index, is an OpenCitations Index containing citations deposited by individuals. We described the rationale and the main ideas behind the development of CROCI in an article published in the proceedings of the 17th International Conference on Scientometrics and Informetrics, held in Rome in September 2019. We described the procedure to follow for adding new citation data into CROCI in the CROCI GitHub repository.
